# Crossword Puzzle Application (v21)

This project is a Python-based Crossword Puzzle Application developed as part of academic project work.

## Features
- Interactive crossword puzzle with Across and Down clues
- Graphical User Interface (GUI) using PyQt6
- Real-time checking of answers
- Scoring system with time tracking
- Leaderboard and admin controls

## Technologies Used
- Python 3
- PyQt6
- pandas
- openpyxl
- Pillow

## Usage
Run the main Python file to start the application:

python v21.py
